# Up to SO(32) Via Supersymmetry Bootstrap

**Author:** Alejandro Rivero
**viXra:** 1901.0074
**Submitted:** January 6, 2019

## Abstract

A simple self-referencing postulate in the Chan-Paton factors of a string allows to fix some of the freedom in the landscape of SO(32) models aspiring to be standard-model-like.

---

## Main Strategy

Boson unoriented string SO(10) Chan-Paton → SO(32) embedding.

### Postulates
- Preonic diquarks charge assignment: "up" (+2/3) and "down" (-1/3)
- **Number parity**: equal up/down diquarks
- **Neutral sector**: charged mesons = neutral mesons

### Self-Consistency Conditions

$$2N = k_u k_d$$

$$2N = \frac{k_d(k_d+1)}{2}$$

$$4N = k_u^2 + k_d^2 - 1$$

**Unique solution:** $N=3$, $k_u=2$, $k_d=3$, $r+s=5$, uniquely fixed by U(1) charge.

---

## Recursive Coloring

$$\mathbf{3 \times 3 = 6 \oplus 3}$$

so a composite 3 is self-composed. The mesonic strings:

$$\mathbf{3 \times \bar{3} = 8 \oplus 1}$$

for mesons.

### Decomposition Path

$$SO(32) \supset SO(30) \supset SO(30) \times SU(3) \supset SO(15) \times U(1) \times SU(3)$$

## "Turtles All the Way Down"

Metaphor for the self-referential composite structure: the preons that label the string are themselves a subset of the quarks produced by the string.
