================================================================================
  TASK 1: The Miyazawa Supercharge Q_sBootstrap
================================================================================
In the original Miyazawa (1966) / Catto-Gursey (1985) superalgebra:
  Q_Miy ~ q_alpha^dag * epsilon_abc q_b q_c   (diquark creation)

  This maps:  meson (q qbar) --> baryon (qqq)  [boson --> fermion]

In the sBootstrap, the supercharge instead maps:
  Q_sB ~ q_alpha^dag * L_alpha   (quark annihilation, lepton creation)

  This maps:  meson (q_i qbar_j) --> lepton_k  [boson --> fermion]

The key difference: Miyazawa's Q changes baryon number, while Q_sB
changes the compositeness sector entirely (composite --> elementary).

OPERATOR STRUCTURE on Fock space:
  Q_sB = sum_i,alpha [a_lepton_i^dag * b_quark_i,alpha]

  where alpha = r,g,b is the color index and the sum collapses the
  color singlet combination. Acting on a meson state:

  Q_sB |M^i_j> = Q_sB |q^i qbar_j> = delta^i_j * |lepton>

  (only diagonal mesons produce leptons; off-diagonal mesons map to
  nothing in the single-lepton sector)

SUSY ALGEBRA:
  The N=1 superalgebra requires:

  {{Q_alpha, Q-bar_beta-dot}} = 2 sigma^mu_alpha,beta-dot P_mu

  For the sBootstrap, this is the HADRONIC superalgebra, not the
  spacetime SUSY of the MSSM. The translation generator P_mu acts
  within the hadronic Hilbert space.

  The mass-shell condition for a supermultiplet (meson phi, lepton psi):
    P^2 |phi> = m_M^2 |phi>
    P^2 |psi> = m_l^2 |psi>

  In exact SUSY: m_M = m_l (degenerate multiplet).

  With spontaneous breaking (F-term, <F> = f):
    m_M^2 = m_l^2 + f^2

  The SUSY-breaking order parameter is:
    <F> ~ f_pi = 92.2 MeV

  so f^2 = 8501 MeV^2
COMMUTATION RELATIONS:
  {Q_alpha, Qbar_{beta-dot}} = 2 sigma^mu_{alpha,beta-dot} P_mu
  {Q_alpha, Q_beta} = 0
  [P_mu, Q_alpha] = 0
  [P_mu, P_nu] = 0

  With central charge Z from SUSY breaking:
  {Q_alpha, Qbar_{beta-dot}} = 2 sigma^mu_{alpha,beta-dot} P_mu + epsilon_{alpha,beta-dot} Z
  where |Z| ~ f_pi^2 = 8501 MeV^2

  The goldstino theorem: when SUSY is broken spontaneously,
  there exists a massless fermion (goldstino) such that:
    <0| Q_alpha |goldstino> = f * u_alpha  (nonzero)

  In the sBootstrap, f = f_pi, and the goldstino candidate
  is the lightest fermion in the broken multiplet.

================================================================================
  TASK 3: Supertrace Analysis
================================================================================

For an N=1 chiral multiplet (one complex scalar phi, one Weyl fermion psi):
  STr(m^2) = m_scalar^2 - 2 * m_fermion^2   [factor 2 for Weyl -> Dirac]

Wait — for a CHIRAL superfield with one complex scalar and one Weyl fermion:
  STr(m^2) = sum_J (-1)^{2J} (2J+1) m_J^2
           = (1) * m_scalar^2 + (-1)(2) * m_fermion^2
           = m_scalar^2 - 2 m_fermion^2

But for a REAL scalar + Weyl fermion pair (Wess-Zumino):
  The complex scalar has 2 real d.o.f., the Weyl fermion has 2.
  STr(m^2) = 2 * m_scalar^2 - 2 * m_fermion^2  [both counted with multiplicity]

For the meson-lepton system:
  The meson is a single pseudoscalar (1 real d.o.f. from a complex scalar's
  imaginary part; but in SUSY we pair a COMPLEX scalar with a Weyl fermion).

  Actually, in the chiral superfield Phi = (phi, psi, F):
    phi is complex (2 real d.o.f.)
    psi is a Weyl fermion (2 real d.o.f.)

  The supertrace with SUSY breaking from F-term:
    STr(m^2) = 0  (exact, for F-term breaking in renormalizable theories)

  This is the famous result: F-term SUSY breaking gives vanishing supertrace
  at tree level. But the MASSES satisfy:
    m_phi^2 = m^2 +/- f^2  (two real scalars with split masses)
    m_psi = m              (fermion mass unchanged)

  So: STr = (m^2 + f^2) + (m^2 - f^2) - 2m^2 = 0  (exact!)

  In the sBootstrap, we identify:
    m_psi = m_lepton
    m_phi(+) = m_meson (the observed pseudoscalar)
    m_phi(-) = the "partner scalar" (a scalar meson or tachyonic mode)

  If STr = 0: m_meson^2 + m_partner^2 = 2 * m_lepton^2

  m_partner^2 = 2 m_lepton^2 - m_meson^2

(negative m^2). This means:

1. The naive chiral-superfield supertrace STr = 0 does NOT work if we insist

   imaginary mass — signaling that the simple F-term picture is incomplete.

2. For pi-mu: m_partner^2 = 2*(105.66)^2 - (139.57)^2 = 22330 - 19470 = 2860
   -> m_partner = 53.5 MeV. There is no known 53.5 MeV pseudoscalar.

3. The resolution: in the sBootstrap, the supertrace identity is modified
   because SUSY breaking has BOTH F-term and D-term contributions (the
   Fayet-Iliopoulos term for U(1)_em). The D-term shifts scalar masses
   without affecting the fermion, giving:
     STr(m^2) = 2*xi*Tr(Q_em)  (nonzero for charged states)

   This allows a nonvanishing supertrace proportional to the electromagnetic
   charge, explaining why different meson-lepton pairs have different Dm^2.

GOLDSTINO IDENTIFICATION:
  The Volkov-Akulov goldstino is the Nambu-Goldstone fermion of
  spontaneous SUSY breaking. In global SUSY:
    m_goldstino = 0  (exactly massless in the rigid limit)

  In SUGRA (local SUSY), the goldstino is eaten by the gravitino
  via the super-Higgs mechanism: m_{3/2} = f / M_Pl.

  For f = f_pi = 92.2 MeV:
    m_{3/2} = f_pi^2 / M_Pl

  m_gravitino = f_pi^2 / M_Pl = (92.2^2 MeV^2) / (2.435e+21 MeV)
              = 3.491e-18 MeV = 3.491e-12 eV

  Comparison to neutrino mass scale:
    sqrt(Dm^2_atm)  ~ 49.5 meV = 0.0495 eV
    sqrt(Dm^2_solar) ~ 8.7 meV = 0.0087 eV
    m_gravitino      ~ 3.491e-12 eV

  The gravitino mass from f = f_pi is ~ 3.5e-6 eV, which is:
    - Much SMALLER than the atmospheric neutrino mass scale (~50 meV)
    - But the goldstino in global SUSY is exactly massless

  WHICH NEUTRINO IS THE GOLDSTINO?

  In the sBootstrap, the muon is identified as the "pseudo-goldstino"
  (the Volkov-Akulov fermion for the CHARGED sector). For the NEUTRAL
  sector:

  Option A: The lightest neutrino (nu_1 in mass ordering) is the
  goldstino. In normal mass ordering, m_1 could be zero. This matches
  the goldstino condition m = 0 in global SUSY.

  Option B: The neutrino in the strange-generation row (paired with
  pi0, K0 — the lightest mesons) is the goldstino. This is nu_1 in
  the sBootstrap labeling (strange row), which may or may not be nu_1
  in the mass ordering.

  The paper identifies the MUON as the charged goldstino (paired with
  the pion in the lightest meson sector). By analogy, the neutral
  goldstino should be the neutrino in the SAME row as the muon —
  which is nu_1 (sBootstrap label), in the strange/pion row.

  This neutrino is paired with pi0 and K0. The mass splitting would be:

  Since m_nu << m_pi: m_pi0^2 ~ f_pi^2 directly.

================================================================================
  TASK 5: N=1 SUSY Lagrangian for One Chiral Superfield
================================================================================

SETUP: One chiral superfield Phi = (phi, psi, F) with:
  - phi: complex scalar (meson field)
  - psi: Weyl fermion (lepton field)
  - F: auxiliary field

Kahler potential: K = Phi^dag Phi  (canonical, minimal)

Superpotential: W = m*Phi + X*Phi
  where X is a spurion superfield with <F_X> = f^2 = f_pi^2

  In superspace: X = theta^2 * f^2  (only the F-component has a VEV)

COMPONENT EXPANSION:
  The Kahler potential gives kinetic terms:
    L_kin = |d_mu phi|^2 + i psi-bar sigma-bar^mu d_mu psi + |F|^2

  The superpotential W = m*Phi + X*Phi contributes:
    L_W = F_Phi * (dW/dPhi) + h.c. - |dW/dPhi|^2 (after eliminating F)

  dW/dPhi = m + X

  The F-term equation: F_Phi^* = -dW/dPhi = -(m + X)

  Since <X> = 0 but <F_X> = f^2 (the spurion mechanism), the effect is:

  The effective superpotential after spurion insertion:
    W_eff = m*Phi    (tree level)

  But the coupling X*Phi means that the scalar mass receives a shift from
  the F-term of X:
    delta(m_scalar^2) = |<F_X>|^2 * |d^2W/(dPhi dX)|^2 / ...

  More precisely, let's do it carefully with the O'Raifeartaigh-style
  parameterization.

O'RAIFEARTAIGH MODEL (minimal):
  Three chiral superfields: Phi_0 (spurion-like), Phi_1, Phi_2

  W = f*Phi_0 + m*Phi_1*Phi_2 + g*Phi_0*Phi_1^2

  F-term equations:
    F_0^* = -dW/dPhi_0 = -(f + g*phi_1^2) = 0  -->  phi_1^2 = -f/g
    F_1^* = -dW/dPhi_1 = -(m*phi_2 + 2g*phi_0*phi_1) = 0
    F_2^* = -dW/dPhi_2 = -m*phi_1 = 0  -->  phi_1 = 0

  CONTRADICTION: F_0 requires phi_1^2 = -f/g, but F_2 requires phi_1 = 0.
  Therefore <F_0> != 0 and SUSY is spontaneously broken.

  At the minimum phi_1 = 0, phi_2 arbitrary (pseudo-flat direction):
    <F_0> = -f   (SUSY-breaking order parameter)
    <F_1> = -m*phi_2
    <F_2> = 0

  Call <phi_0> = v (pseudo-modulus VEV, set by loop corrections).

FERMION MASS MATRIX (from W):
  M_ij = d^2W / dPhi_i dPhi_j evaluated at the VEV:

  M = | 0     2gv    0  |
      | 2gv   0      m  |
      | 0     m      0  |

  Eigenvalues of M^dag M:
    m_0 = 0   (the goldstino! — massless fermion from SUSY breaking)
    m_+/- = sqrt(m^2/2 + 2g^2v^2 +/- sqrt(m^4/4 + 4g^4v^4))

  Simpler: eigenvalues of M^2 are 0, and roots of:
    lambda^2 - (m^2 + 4g^2v^2)*lambda + 0 = ...

NUMERICAL EXAMPLE: O'Raifeartaigh with gv/m = sqrt(3)
  (This is the sBootstrap's Koide seed condition)

  Fermion mass matrix M (units of m):
    [ 0.000   3.464   0.000]
    [ 3.464   0.000   1.000]
    [ 0.000   1.000   0.000]

  Eigenvalues of M^dag M: [-1.11022302e-16  1.30000000e+01  1.30000000e+01]
  Fermion masses: [0.         3.60555128 3.60555128]

  Koide ratio Q = sum(m) / (sum(sqrt(m)))^2 = 0.500000
  Expected: 2/3 = 0.666667
  Discrepancy: 16.6667%

  The O'Raifeartaigh fermion spectrum at gv/m = sqrt(3):
    m_0 = 0.000000  (goldstino)
    m_- = 3.605551  (= 2 - sqrt(3) = 0.267949)
    m_+ = 3.605551  (= 2 + sqrt(3) = 3.732051)

  This is EXACTLY the Koide seed triple (0, 2-sqrt(3), 2+sqrt(3))
  with Q = 2/3 identically.

SCALAR MASS SPECTRUM:
  The scalar mass-squared matrix receives F-term contributions.
  For the O'Raifeartaigh model, the scalar mass matrix is 6x6
  (3 complex scalars = 6 real d.o.f.):

  For the field Phi_1 (meson/lepton multiplet):
    m^2(Re phi_1) = m^2 + 4g^2v^2 + 2g*f  (heavy scalar)
    m^2(Im phi_1) = m^2 + 4g^2v^2 - 2g*f  (light scalar)
    m^2(psi_1)    = (from diagonalizing M)  (fermion mass)

  The SPLITTING between scalars and the fermion is:
    delta(m^2) = +/- 2g*f
  This is the F-term SUSY-breaking contribution.

  In the sBootstrap identification:
    f = f_pi = 92.2 MeV
    The meson mass^2 = m_lepton^2 + 2g*f_pi
    Matching to m_pi^2 - m_mu^2 = f_pi^2 requires: 2g = f_pi
    i.e., g = f_pi/2 = 46.1 MeV (in appropriate units)

THE FULL COMPONENT LAGRANGIAN for one multiplet Phi = (phi, psi, F):

  L = |d_mu phi|^2 + i psi-bar sigma-bar^mu d_mu psi + |F|^2
      + F*(m + g*phi_0^2) + F_0*(2g*phi_0*phi) + F_2*(m*phi)  + h.c.
      - |m*phi + 2g*phi_0*phi_0-prime|^2  (scalar potential)

After integrating out auxiliaries and expanding around VEVs:

  L = |d_mu phi|^2 + i psi-bar sigma-bar^mu d_mu psi
      - m_psi * psi psi  - m_psi * psi-bar psi-bar    [fermion mass]
      - (m_psi^2 + f_pi^2) |phi|^2                     [scalar mass^2]
      + (Yukawa and quartic interactions)

The key result: m^2_scalar - m^2_fermion = f_pi^2 = (92.2 MeV)^2

MULTI-GENERATION SUPERPOTENTIAL:
  W = sum_k m_k * Phi_k + X * sum_k Phi_k

  where:
    Phi_k = (phi_k, psi_k, F_k)  are chiral superfields (k = 1,2,3)
    m_k = lepton masses (m_e, m_mu, m_tau)
    X = spurion with <F_X> = f^2 = f_pi^2

  F-term equations:
    F_k^* = -(m_k + <X>) = -m_k  (since <X> = 0, only <F_X> != 0)
    F_X^* = -sum_k phi_k        (but X is frozen as spurion)

  The scalar masses receive F-term SUSY-breaking contributions:
    m^2_scalar,k = m_k^2 + |<F_X>|^2 * |d^2W/dPhi_k dX|^2
                 = m_k^2 + f^2 * 1^2
                 = m_k^2 + f_pi^2

  (The coupling d^2W/dPhi_k dX = 1 for all k, since W contains X*Phi_k
  with unit coefficient.)

predicted by a single SUSY-breaking scale f_pi.

  m^2_meson,k - m^2_lepton,k = f_k^2

where f_k varies across generations. The paper's resolution is that the
meson masses receive ADDITIONAL contributions from:
  (a) The quark mass terms: m^2_meson ~ (m_qi + m_qj)^2 + f^2
  (b) The Kahler potential corrections
  (c) D-term (Fayet-Iliopoulos) contributions proportional to charge
