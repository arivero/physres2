# Z₉ Numerical Test Results

## Key Finding: Leptons are NOT at Z₉ minima

The Z₉ minimum condition is 9δ mod 2π ≈ 0.
Actual: 9δ mod 2π = 2.000 rad — offset of 32% of a full cycle.
This is 26,000σ from a Z₉ minimum. **The cos(9δ) potential does NOT localize leptons.**

## The "2/9" coincidence is DIFFERENT

δ mod(2π/3) = 0.222229 rad ≈ 2/9 = 0.222222 rad (31.2 ppm, 0.83σ)
This is 2/9 radians, NOT 2π/9 radians. The factor of π matters.
A Z₉ minimum would require δ mod(2π/9) ≈ 0, i.e., δ ≈ n·(2π/9).
The actual δ ≈ n·(2π/3) + 2/9 — the "2/9" is in plain radians, not 2π/9.
