# Z₃ Orbifold Yukawa Couplings — Results

## Fixed points on T²(SU(3) root lattice)
- f₁ = 0, f₂ = (e₁+e₂)/3, f₃ = 2f₂ (collinear!)
- Verified: ωf₂ - f₂ = -e₁ (lattice vector)

## Minimal instanton triangle
- Fixed points are collinear → must use lattice-shifted copies
- A_min = √3/12 ≈ 0.144 (area of equilateral triangle, side 1/√3)
- 6 such triangles tile the fundamental domain (3 up + 3 down)

## Instanton action (hadronic string, α' ~ 1 GeV⁻²)
- S = A_min/(2πα') ≈ √3/(24π) ≈ 0.023
- g/m ~ exp(-0.023) ≈ 0.977 (NOT exponentially suppressed)

## Ratio gv/m = √3
- With g/m ~ 0.977, need v/m ~ √3/0.977 ≈ 1.78 (O(1), no tuning)
- √3 = |e₁+e₂| = lattice diagonal = 2 A_fund = 3 × (fixed-point spacing)
- √3 is a geometric natural scale of the SU(3) root lattice
- But dynamics (nonperturbative stabilization) must select it

## High string scale (M_s ~ M_GUT)
- S ~ R²/α' >> 1 → g exponentially small
- gv/m = √3 requires exponentially large v/m → unnatural
- Hadronic/ISS context (α' ~ 1 GeV⁻²) avoids this
