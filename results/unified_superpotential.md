# Unified Superpotential Structure

## The Full Superpotential

W = Tr(m̂ M) + X(det M^(3) − BB̄ − Λ⁶) + y_c H M^c_c + y_b H M^b_b

(chiral limit m_u = m_d → 0 for light sector)

## Layered Structure

### Layer 1: SQCD/F-term
Seiberg constraint with m_s ≠ 0 forces SUSY breaking.
Effective theory is O'Raifeartaigh-type with pseudo-moduli M_uu, M_dd.
In chiral limit, ADS runaway; ISS deformation stabilizes with F ≠ 0.

### Layer 2: Koide Seed
h = √3 in magnetic dual (from gv/m = √3).
Fixes mass ratio m_c/m_s = (2+√3)²/(2−√3)² = 13.93.
Prediction: m_c = 1301 MeV (PDG: 1270, 2.4% deviation).

### Layer 3: Bloom / v₀-doubling
√m_b = 3√m_s + √m_c → m_b = 4177 MeV (0.1σ from PDG).
**Non-holomorphic**: z_k = √m_k ~ 1/√⟨M^k_k⟩ is non-holomorphic.
Cannot come from superpotential. Must be Kähler potential,
nonperturbative effects, or emergent dynamics.

### Layer 4: Yukawa/EW
(c,b,t) triple governed by Higgs Yukawa couplings.
Q(c,b,t) ≈ 2/3 is an overdetermination test, not independent constraint.
