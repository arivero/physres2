# The String Behind the sBootstrap: Toward a Type I Construction
## A. Rivero, with analysis by Claude — Working notes, March 2026

### Construction summary

Type IIB on R^{3,1} × T^6/Z_3 with orientifold Ω

- Internal space: T^6 = (T^2)^3 with each T^2 at SU(3) root lattice (τ = e^{iπ/3})
- Orbifold: Z_3 acting as (z_1, z_2, z_3) → (ωz_1, ωz_2, ωz_3), ω = e^{2πi/3}
- Fixed points: 27 (= 3^3)
- CP embedding: γ_θ = diag(ω·1_5, ω²·1_5, 1·1_5, 1) ∈ U(16)

Branes:
- 16 D9-brane pairs (32 half-branes)
  - 5 × 3 = 15 at generic points (5 flavors × 3 Z_3 images) → U(5)_f
  - 1 at fixed point (Z_3-invariant) → top/singlet
- 3 D5-branes at a fixed point → SU(3)_c
- O9+ plane → symmetric diquarks (15)
- O5 planes at fixed points → tadpole cancellation

Matter:
- 9-9 strings → mesons (24 ⊕ 1 of SU(5)_f)
- 5-9 strings → quarks ((3, 5) of SU(3)_c × SU(5)_f)
- 9-9 unoriented → diquarks (15 of SU(5)_f, color 3̄)
- 5-5 strings → gluons, gluinos

### Key automatic features
1. Three generations from Z_3 images
2. Symmetric 15 from O+ projection
3. SU(5)_f from D9 stack
4. SU(3)_c from D5 stack
5. Top excluded from composites (fixed-point brane)
6. gv/m = √3 conjectured from SU(3) lattice geometry (A_min = √3/4)
7. cos(9δ) from Z_3 fractional instantons
8. Full Regge tower from string excitations
