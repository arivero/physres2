# Kähler Potential and v₀-Doubling

## Key Results

### v₀-doubling is non-perturbative
- Canonical Kähler K = Tr(M†M) gives NO v₀-doubling
- Non-canonical (Tr M†M)² / Λ⁶ correction has NO effect on VEVs
- CW potential generates smooth functions — CANNOT produce |Tr(√M)|²
- The √M singularity in Kähler metric signals nonperturbative origin

### Formal Kähler home
The term K ⊃ c₁ |Tr(√M)|² / Λ can accommodate v₀-doubling:
- Works in magnetic dual frame where M_i ∝ m_i
- Has 1/√M singularity (non-perturbative)
- Formal, not dynamical — no known mechanism generates it

### Overconstraining (IMPORTANT)
The three constraints are mutually inconsistent:
- **Seed Koide**: m_c/m_s = (2+√3)² = 13.93 → m_c = 1301 MeV
- **v₀-doubling**: √m_b = 3√m_s + √m_c → m_b = 4233 MeV (with exact seed m_c)
- **Overlap prediction**: Q(-s,c,b) = Q(c,b,t) = 2/3 → m_b = 4159 MeV

With PDG m_c = 1270: v₀-doubling gives m_b = 4177 (0.1σ) ✓
With exact seed m_c = 1301: v₀-doubling gives m_b = 4233 (1.8σ) ✗
With overlap prediction: v₀ ratio = 1.968 (not 2) ✗

The physical system uses PDG m_c ≠ exact seed m_c. The 2.4% charm deviation
from exact seed is what makes v₀-doubling work well with physical masses.

### Simultaneously requiring Q=2/3 + v₀=2
Requires m_c/m_s = ((-1+3√5)/2)² = 8.15, which conflicts with
seed Koide value 13.93. The constraints are overconstrained.

### Frame dependence
- Electric frame: ⟨M_i⟩ ∝ 1/m_i (Seiberg seesaw) — v₀-doubling absent
- Magnetic frame: ⟨M̃_i⟩ ∝ m_i — v₀-doubling natural
- v₀-doubling lives in the magnetic description
