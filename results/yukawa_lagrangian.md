## Part (a): Light Confined Sector (u, d, s)

The mass term in the superpotential is

Tr(m_hat M) = m_u M^u_u + m_d M^d_d + m_s M^s_s

Since the current quark masses are generated by the Higgs mechanism with the type-II
two-Higgs-doublet assignment (up-type quarks couple to H_u, down-type to H_d):

m_u = y_u v_u    (u is up-type)
m_d = y_d v_d    (d is down-type)
m_s = y_s v_d    (s is down-type)

Substituting:

Tr(m_hat M) = y_u v_u M^u_u + y_d v_d M^d_d + y_s v_d M^s_s

Factoring out the Higgs VEVs (before EWSB, replacing v_{u,d} by the Higgs fields):

Tr(m_hat M) = H_u (y_u M^u_u) + H_d (y_d M^d_d + y_s M^s_s)

Or equivalently, writing m_i = y_i v_type(i):

Tr(m_hat M) = (v_u / v_u)(m_u M^u_u) + (v_d / v_d)(m_d M^d_d + m_s M^s_s)
            = (1/v_u)[v_u m_u M^u_u] + (1/v_d)[v_d (m_d M^d_d + m_s M^s_s)]

At tan beta = 1 where v_u = v_d = v/sqrt(2):

Tr(m_hat M) = (sqrt(2)/v) [m_u M^u_u + m_d M^d_d + m_s M^s_s]

times the appropriate Higgs field. The mass term IS the Yukawa coupling — no
separate Yukawa vertex is needed for the light confined quarks because the meson
fields M^j_j are the dynamical degrees of freedom and the current quark masses m_j
already encode the Higgs coupling.

Explicit Yukawa couplings for the light sector:

| d | m_d / v_d | 2.6823 x 10^{-5} |
| s | m_s / v_d | 5.3646 x 10^{-4} |

## Part (b): Charm and Bottom

The explicit Yukawa terms in the superpotential are

y_c H_u M^c_c + y_b H_d M^b_b

where M^c_c = Q^c Q-bar_c and M^b_b = Q^b Q-bar_b are elementary bilinears above
their respective thresholds.

### Yukawa couplings at tan beta = 1

y_c = m_c / v_u = sqrt(2) m_c / v = 1275 / 174103.83 = **7.3232 x 10^{-3}**

y_b = m_b / v_d = sqrt(2) m_b / v = 4180 / 174103.83 = **2.4009 x 10^{-2}**

### Seesaw-vacuum product y_j <M_j>

If the seesaw extends to charm and bottom (imagining them as part of the confined
sector), the meson VEVs would be M^c_c = C/m_c, M^b_b = C/m_b. Then:

y_c <M^c_c> = (m_c / v_u)(C / m_c) = C / v_u = 5.0677 MeV

y_b <M^b_b> = (m_b / v_d)(C / m_b) = C / v_d = 5.0677 MeV

Both equal C/v_type. The quark mass m_j cancels exactly in the product y_j M_j.
This is the **flavor-universal seesaw relation**: at the seesaw vacuum,
y_j M_j = C / v_type is independent of the quark flavor j.

At tan beta = 1 (v_u = v_d):

y_j <M_j> = C / v_u = sqrt(2) C / v = **5.0677 MeV** for all j

## Part (c): Top Quark

The top quark is elementary (not confined): the superpotential term is

y_t H_u Q^t Q-bar_t

After EWSB, H_u -> v_u, giving mass m_t = y_t v_u. Therefore:

y_t = m_t / v_u = 172760 / 174103.83 = **0.9923**

This is y_t = sqrt(2) m_t / v = sqrt(2) x 0.7016 = 0.9923 ≈ **1**.

**Regarding the claim y_t ≈ sqrt(2):** In the component-field convention where the
Higgs doublet is expanded as H = (v + h)/sqrt(2) (so the doublet VEV is v/sqrt(2),
not v), the Yukawa coupling in the Lagrangian is y_t psi_Q H psi_t and the mass
relation becomes m_t = y_t v_u / sqrt(2). In that convention:

y_t = sqrt(2) m_t / v_u = 2 m_t / v = 2 x 172760 / 246220 = **1.4033**

which is within 0.8% of sqrt(2) = 1.4142. The near-equality y_t ≈ sqrt(2)
reflects the empirical fact that m_t ≈ v/sqrt(2) = v_u ≈ 174 GeV.

In the superpotential convention used throughout this document (m = y v_u without
extra factors), y_t = 0.992 ≈ 1.

## Part (d): Fermion Mass Matrix from the Superpotential

The fermion (mesino) mass matrix is W_{IJ} = d^2 W / d Phi_I d Phi_J evaluated at
the seesaw vacuum.

### Lagrange multiplier value X_0

From the F-term equation dW/dM^i_i = 0:

m_i + X_0 * (det M / M^i_i) = 0

Since det M / M^i_i = Lambda^6 / M_i and M_i = C/m_i:

X_0 = -m_i M_i / Lambda^6 = -C / Lambda^6

This gives the same X_0 from all three flavors (consistency check):

X_0 = -C / Lambda^6 = -882297.43 / 7.290 x 10^{14} = **-1.2103 x 10^{-9} MeV^{-4}**

Verification:
- From u: X_0 = -m_u / (Lambda^6/M_u) = -2.16 / (1.7847 x 10^9) = -1.2103 x 10^{-9}
- From d: X_0 = -m_d / (Lambda^6/M_d) = -4.67 / (3.8586 x 10^9) = -1.2103 x 10^{-9}
- From s: X_0 = -m_s / (Lambda^6/M_s) = -93.4 / (7.7172 x 10^{10}) = -1.2103 x 10^{-9}

All three agree to floating-point precision.

### Diagonal mesinos: 4x4 central block {M^u_u, M^d_d, M^s_s, X}

From Tr(m_hat M): d^2/dM^i_i dM^j_j = 0 (linear in M).

From X det M: d^2(det M)/dM^i_i dM^j_j = M_k for i != j != k (the third
diagonal VEV). The mixed derivatives dM^i_i dX = det M / M_i = Lambda^6 / M_i.

The 4x4 mass matrix (in the basis {M^u_u, M^d_d, M^s_s, X}) is:

|        | M^u_u     | M^d_d     | M^s_s     | X              |
|--------|-----------|-----------|-----------|----------------|
| M^u_u  | 0         | X_0 M_s   | X_0 M_d   | Lambda^6/M_u   |
| M^d_d  | X_0 M_s   | 0         | X_0 M_u   | Lambda^6/M_d   |
| M^s_s  | X_0 M_d   | X_0 M_u   | 0         | Lambda^6/M_s   |
| X      | Lambda^6/M_u | Lambda^6/M_d | Lambda^6/M_s | 0       |

Numerically:

|        | M^u_u     | M^d_d     | M^s_s     | X              |
|--------|-----------|-----------|-----------|----------------|
| M^u_u  | 0         | -1.143e-5 | -2.287e-4 | +1.785e+9      |
| M^d_d  | -1.143e-5 | 0         | -4.944e-4 | +3.859e+9      |
| M^s_s  | -2.287e-4 | -4.944e-4 | 0         | +7.717e+10     |
| X      | +1.785e+9 | +3.859e+9 | +7.717e+10| 0              |

Eigenvalues of the 4x4 block:

| Mode | Eigenvalue |
|------|------------|
| Heavy + | +7.7289 x 10^{10} |
| Light + | +5.2459 x 10^{-5} |
| Light + | +7.5476 x 10^{-6} |
| Heavy - | -7.7289 x 10^{10} |

The two large eigenvalues +/- 7.73 x 10^{10} correspond to the X-dominated mode
(its mass scale is Lambda^6/M_s ~ 7.7 x 10^{10}). The two small eigenvalues are
the residual mesino masses after X is integrated out.

### Off-diagonal mesinos: three 2x2 blocks

For each off-diagonal pair (M^a_b, M^b_a) with a != b, the only nonzero second
derivative comes from X_0 det M:

d^2(X det M) / dM^a_b dM^b_a = X_0 M_k

where k is the third flavor (neither a nor b). This gives a 2x2 matrix
[[0, X_0 M_k], [X_0 M_k, 0]] with eigenvalues +/- |X_0 M_k|.

| Pair (a,b) | Third flavor k | Mass = |X_0| M_k (MeV) |
|------------|---------------|-------------------------|
| (u, d)     | s             | 1.143 x 10^{-5}        |
| (u, s)     | d             | 2.287 x 10^{-4}        |
| (d, s)     | u             | 4.944 x 10^{-4}        |

Note the **inverse hierarchy**: the off-diagonal mesino associated with the heaviest
quark (strange) gets the smallest mass, because M_s = C/m_s is the smallest VEV.

## Part (e): Mesino Mass Hierarchy and X_0

### What determines X_0?

X has no kinetic term, so it acts as a Lagrange multiplier. Its equation of motion is

dW/dX = det M - B B-tilde - Lambda^6 = 0

This equation constrains M and B but does NOT determine X_0 itself. Rather,
X_0 is fixed by the other F-term equations. From dW/dM^i_i = 0:

m_i + X_0 (Lambda^6 / M_i) = 0 =>  X_0 = -m_i M_i / Lambda^6 = -C / Lambda^6

So X_0 is determined, and all mesino masses are proportional to |X_0|:

- Off-diagonal: |X_0| M_k
- Diagonal: from diagonalizing the 4x4 block above

### Are the mesino masses physical?

Since X is an auxiliary field with no propagating degree of freedom, the terms
proportional to X_0 in the fermion mass matrix arise from the constraint structure.
Integrating out X by its equation of motion:

F_X = dW/dX = det M - B B-tilde - Lambda^6 = 0

imposes the quantum modified moduli space constraint. The fermionic partner of X
(the "X-ino") has no kinetic term either, so it too must be integrated out. After
integrating out the X-ino, the effective mesino mass matrix receives corrections.

At tree level in the Seiberg dual, the off-diagonal mesino masses |X_0| M_k are
extremely small (~10^{-5} to ~10^{-4} MeV). These tiny masses reflect the fact that
the pseudomoduli are almost flat directions. Physical mesino masses would be lifted
by:

1. **Coleman-Weinberg (CW) corrections** from the Kahler potential, once X is
   properly integrated out
2. **SUSY-breaking soft masses**, which split scalar and fermion meson components
3. **Electroweak corrections** once the Higgs Yukawa couplings are turned on

The CW potential from the bion Kahler correction and the X quartic K_X = -|X|^4/(12 mu^2)
generates masses of order g v / (16 pi^2) that can dominate over the tree-level values.

## Part (f): Flavor-Universal Yukawa Relation

At the seesaw vacuum, for each confined quark j:

y_j <M_j> = (m_j / v_type) x (C / m_j) = C / v_type

The quark mass m_j cancels exactly. This is true for ANY quark flavor, giving:

**y_j M_j = C / v_type  (flavor-universal)**

At tan beta = 1 (v_u = v_d = v/sqrt(2)):

y_j M_j = C / (v/sqrt(2)) = sqrt(2) C / v

Numerically:

y_j <M_j> = sqrt(2) x 882297.43 / 246220 = **5.068 MeV** for all j

Explicit verification:

| Quark j | y_j | M_j (MeV) | y_j M_j (MeV) |
|---------|-----|-----------|---------------|
| u | 1.241 x 10^{-5} | 408471.0 | 5.068 |
| d | 2.682 x 10^{-5} | 188928.8 | 5.068 |
| s | 5.365 x 10^{-4} | 9446.4   | 5.068 |
| c | 7.323 x 10^{-3} | 692.0    | 5.068 |
| b | 2.401 x 10^{-2} | 211.1    | 5.068 |

### Physical interpretation

The flavor-universality of y_j M_j means that at the seesaw vacuum, the effective
mass contribution from the Yukawa coupling times the meson VEV is the SAME for all
confined quarks, regardless of their actual mass. This is a direct consequence of
the seesaw: heavy quarks couple weakly (small y_j) but have large meson VEVs
(large M_j = C/m_j), and these two effects cancel exactly.

Physically, this means:
1. The seesaw vacuum is an **equal-coupling point** — all mesons couple to the Higgs
   with the same effective strength C/v_type.
2. The quark mass hierarchy is entirely encoded in the Yukawa couplings y_j, not in
   the meson VEVs. The VEVs run inversely to the masses.
3. The universal scale sqrt(2) C/v = 5.068 MeV = 0.00567 Lambda represents the
   ratio of the confinement scale squared to the electroweak scale.

## Part (g): Complete Yukawa Coupling Matrix

### Up-type Yukawa matrix Y^u (coupling to H_u)

In the basis (u, c, t), the Yukawa matrix is diagonal at tree level:

Y^u = diag(y_u, y_c, y_t) = diag(1.241 x 10^{-5}, 7.323 x 10^{-3}, 0.992)

where:
- y_u = m_u / v_u = 1.2406 x 10^{-5}  (confined sector, from Tr(m_hat M))
- y_c = m_c / v_u = 7.3232 x 10^{-3}  (elementary Yukawa y_c H_u M^c_c)
- y_t = m_t / v_u = 0.9923             (elementary Yukawa y_t H_u Q^t Q-bar_t)

### Down-type Yukawa matrix Y^d (coupling to H_d)

In the basis (d, s, b):

Y^d = diag(y_d, y_s, y_b) = diag(2.682 x 10^{-5}, 5.365 x 10^{-4}, 2.401 x 10^{-2})

where:
- y_d = m_d / v_d = 2.6823 x 10^{-5}  (confined sector, from Tr(m_hat M))
- y_s = m_s / v_d = 5.3646 x 10^{-4}  (confined sector, from Tr(m_hat M))
- y_b = m_b / v_d = 2.4009 x 10^{-2}  (elementary Yukawa y_b H_d M^b_b)

### Structure of the full 6x6 matrix

The full Yukawa coupling matrix in the (u, d, s, c, b, t) basis, separated by type:

```
Y^u (3x3, up-type quarks coupling to H_u):

         u              c              t
u  [ 1.241e-5          0              0     ]
c  [    0          7.323e-3           0     ]
t  [    0              0           0.9923   ]

Y^d (3x3, down-type quarks coupling to H_d):

         d              s              b
d  [ 2.682e-5          0              0     ]
s  [    0          5.365e-4           0     ]
b  [    0              0          2.401e-2  ]
```

**All entries are diagonal** — there is no flavor mixing from the Yukawa sector alone
in this framework. The CKM matrix would need to arise from a separate mechanism
(misalignment between Yukawa eigenstates and mass eigenstates in the confined sector,
or from off-diagonal meson VEVs, or from radiative corrections).

### Three sectors

The 6 quarks divide into three sectors with distinct Yukawa structures:

| Sector | Quarks | Mechanism | Yukawa range |
|--------|--------|-----------|-------------|
| Confined | u, d, s | Tr(m_hat M): mass term IS Yukawa | 10^{-5} to 5 x 10^{-4} |
| Intermediate | c, b | Explicit y_j H M^j_j | 7 x 10^{-3} to 2.4 x 10^{-2} |
| Elementary | t | Explicit y_t H_u Q^t Q-bar_t | 0.99 |

The confined-sector Yukawas are tiny because the light quarks have masses far below
Lambda: y_j ~ m_j / v << 1. The intermediate sector has perturbative Yukawas.
The top Yukawa is O(1), reflecting m_t ~ v_u.

### Koide constraints on the Yukawa eigenvalues

Since Y^u = diag(m_u, m_c, m_t) / v_u and Y^d = diag(m_d, m_s, m_b) / v_d, the
Koide ratio of the Yukawa eigenvalues equals the Koide ratio of the corresponding
quark masses (the common denominator v_u or v_d cancels in the ratio).

| Triplet | Q = (sum m_i) / (sum sqrt(m_i))^2 |
|---------|-----------------------------------|
| (m_u, m_c, m_t) | 0.8488 |
| (m_d, m_s, m_b) | 0.7314 |
| (m_e, m_mu, m_tau) | 0.6667 (= 2/3 to 0.91 sigma) |
| (-sqrt(m_s), sqrt(m_c), sqrt(m_b)) | 0.6745 |
| (m_c, m_b, m_t) | 0.6693 |

The standard up-type and down-type triplets do not individually satisfy Koide (Q = 2/3).
However, the **cross-type** triplets (c, b, t) and (-s, c, b) — which mix up-type and
down-type quarks — satisfy Koide to within a few percent. In the sBootstrap framework,
these are the dynamically selected Koide triples, and the Koide condition constrains
combinations of Yukawa eigenvalues across the up/down sectors rather than within them.

### The mass-Yukawa duality

The seesaw creates a duality between mass and VEV:

m_j  <->  M_j = C / m_j

A heavy quark has a small meson VEV. The Yukawa coupling is proportional to the mass
(y_j = m_j / v_type), while the meson VEV is inversely proportional. Their product
is flavor-universal: y_j M_j = C/v_type = 5.068 MeV.

This means the Yukawa eigenvalues carry ALL the flavor information. The seesaw vacuum
structure (through the constraint det M = Lambda^6) ensures that the meson sector
reorganizes itself to maintain this universal coupling, regardless of the individual
quark mass values.

## Summary: Complete Yukawa Lagrangian

After EWSB (H_u -> v_u + h_u, H_d -> v_d + h_d), the complete Yukawa interaction
Lagrangian in the fermion sector is:

```
L_Yukawa = - y_u (v_u + h_u) psi_{M^u_u} psi_{M^u_u}  (confined)
           - y_d (v_d + h_d) psi_{M^d_d} psi_{M^d_d}  (confined)
           - y_s (v_d + h_d) psi_{M^s_s} psi_{M^s_s}  (confined)
           - y_c (v_u + h_u) psi_{M^c_c} psi_{M^c_c}  (intermediate)
           - y_b (v_d + h_d) psi_{M^b_b} psi_{M^b_b}  (intermediate)
           - y_t (v_u + h_u) psi_{Q^t} psi_{Q-bar_t}   (elementary)
           + h.c.
```

with the mesino mass terms from the constraint sector:

```
L_mass,mesino = - X_0 M_k  psi_{M^a_b} psi_{M^b_a}  (off-diagonal, a != b)
               - (4x4 block terms for diagonal mesinos coupled to X-ino)
               + h.c.
```
